Housing Disassembly

Step 1 of 8: Primary Access

Locate the four M12 retaining bolts on the outer circumference of the pump housing. Using a cross-pattern sequence, loosen each bolt by a quarter turn to ensure even tension release across the housing gasket.

1.1

Remove the protective dust cap from the drive shaft interface using the seal pick tool.

1.2

Mark the housing and the manifold body with a paint pen to ensure correct alignment during reassembly.

1.3

Fully extract the bolts and set them aside in a clean magnetic parts tray.
